specialised storage:
Firmware RAID - A firmware device is a type of device that has hooks in the BIOS allowing it to be used to boot the operating system, if you chooses.

Multipath devices - These provide multiple paths between the computer and its storage devices. They are aggregated and appear to be a singel device to the system. Connections can be provided by iSCSI, or fibre channel of ethernet (FCoE).

Other SAN devices - Any device representing a storage area network (SAN).
