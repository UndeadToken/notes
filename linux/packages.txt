# SSH
Configuration file: /etc/ssh/sshd_config
The ssh service(openssh-server) is disabled by default and runs on port 22.
Each ssh server has its own ssh host keys which are stored in /etc/ssh/ssh_host_*.

enable: sudo systemctl enable ssh && sudo systemctl start ssh
reload: systemctl reload ssh.

Replace pre-generated host keys:
passwd
rm /etc/ssh/ssh_host_*
dpkg-reconfigure openssh-server
service ssh restart
