### 1. What is the maximum size of a single S3 object?
A. 1 TB
B. 5 TB
C. 10 TB
D. Unlimited

**Answer:** B
**Explanation:** S3 objects can be up to 5 TB in size.

---

### 2. For objects larger than 5 GB, which upload method is recommended?
A. Single PUT operation
B. Multi-part upload
C. Transfer Acceleration
D. Batch upload

**Answer:** B
**Explanation:** Multi-part upload is recommended for objects >100 MB and required for objects >5 GB.

---

### 3. Which S3 storage class provides the lowest cost for archival data with retrieval times of 12 hours?
A. Glacier Instant Retrieval
B. Glacier Flexible Retrieval
C. Glacier Deep Archive
D. Standard-IA

**Answer:** C
**Explanation:** Glacier Deep Archive is the lowest cost, with standard retrieval in 12 hours.

---

### 4. What is the default encryption method when SSE-S3 is enabled?
A. RSA-2048
B. AES-128
C. AES-256
D. AES-512

**Answer:** C
**Explanation:** SSE-S3 uses AES-256 encryption with AWS-managed keys.

---

### 5. Which S3 feature is required for Cross-Region Replication (CRR)?
A. Versioning enabled on both source and destination
B. Transfer Acceleration
C. Lifecycle policies
D. Bucket policies

**Answer:** A
**Explanation:** Versioning must be enabled on both source and destination buckets for CRR.

---

### 6. A bucket policy denies all access, but an IAM policy allows s3:GetObject. What is the result?
A. Access allowed
B. Access denied
C. Depends on the principal
D. Error

**Answer:** B
**Explanation:** Explicit DENY always overrides ALLOW in policy evaluation.

---

### 7. What is the maximum duration for an S3 pre-signed URL?
A. 1 hour
B. 12 hours
C. 7 days
D. 1 year

**Answer:** C
**Explanation:** Pre-signed URLs can be valid for up to 7 days (604,800 seconds).

---

### 8. Which S3 storage class automatically moves objects between access tiers based on usage patterns?
A. Standard
B. Standard-IA
C. Intelligent-Tiering
D. One Zone-IA

**Answer:** C
**Explanation:** Intelligent-Tiering automatically moves objects between frequent and infrequent access tiers.

---

### 9. What is the baseline performance for S3 GET requests per prefix?
A. 1,000 requests/second
B. 3,500 requests/second
C. 5,500 requests/second
D. 10,000 requests/second

**Answer:** C
**Explanation:** S3 supports 5,500 GET/HEAD requests per second per prefix.

---

### 10. Which S3 feature uses CloudFront edge locations to accelerate uploads?
A. Multi-part upload
B. Transfer Acceleration
C. Byte-range fetches
D. Cross-Region Replication

**Answer:** B
**Explanation:** S3 Transfer Acceleration uses CloudFront edge locations to speed up uploads over long distances.

---

### 11. An S3 bucket receives ObjectCreated events. Which service can be notified?
A. Lambda only
B. SNS only
C. SQS only
D. Lambda, SNS, or SQS

**Answer:** D
**Explanation:** S3 Event Notifications can trigger Lambda functions, SNS topics, or SQS queues.

---

### 12. Which encryption method requires the client to manage encryption keys?
A. SSE-S3
B. SSE-KMS
C. SSE-C
D. Client-side encryption

**Answer:** C
**Explanation:** SSE-C (Server-Side Encryption with Customer-provided keys) requires the client to provide and manage keys.

---

### 13. What is the durability of S3 Standard storage class?
A. 99.9%
B. 99.99%
C. 99.999999999% (11 nines)
D. 100%

**Answer:** C
**Explanation:** S3 Standard provides 99.999999999% (11 nines) durability across multiple AZs.

---

### 14. Which S3 feature allows transforming objects on retrieval (e.g., redaction, resizing)?
A. S3 Batch Operations
B. S3 Object Lambda
C. Lambda@Edge
D. S3 Select

**Answer:** B
**Explanation:** S3 Object Lambda transforms objects during retrieval using Lambda functions.

---

### 15. A bucket has versioning enabled. What happens when an object is deleted without specifying a version?
A. Object is permanently deleted
B. A delete marker is added
C. All versions are deleted
D. Error is returned

**Answer:** B
**Explanation:** Deleting without version ID adds a delete marker; previous versions remain.

---

### 16. Which S3 storage class stores data in a single Availability Zone?
A. Standard
B. Standard-IA
C. One Zone-IA
D. Glacier

**Answer:** C
**Explanation:** One Zone-IA stores data in a single AZ (lower cost, lower availability).

---

### 17. What is the minimum storage duration charge for S3 Standard-IA?
A. No minimum
B. 30 days
C. 90 days
D. 180 days

**Answer:** B
**Explanation:** Standard-IA has a minimum storage duration of 30 days.

---

### 18. Which HTTP header must be set to enable CORS for S3 static website hosting?
A. Access-Control-Allow-Origin
B. Access-Control-Allow-Methods
C. Access-Control-Allow-Headers
D. All of the above

**Answer:** D
**Explanation:** CORS requires Access-Control-Allow-Origin, Access-Control-Allow-Methods, and Access-Control-Allow-Headers.

---

### 19. An application needs to read only specific columns from a large CSV file in S3. Which feature optimizes this?
A. Byte-range fetches
B. S3 Select
C. Multi-part download
D. Transfer Acceleration

**Answer:** B
**Explanation:** S3 Select retrieves only required data using SQL expressions, reducing data transfer.

---

### 20. Which S3 replication type copies objects within the same region?
A. Cross-Region Replication (CRR)
B. Same-Region Replication (SRR)
C. Multi-Region Replication (MRR)
D. Local Replication (LR)

**Answer:** B
**Explanation:** Same-Region Replication (SRR) replicates objects within the same AWS region.
