Capitalized letters denote public visibility and can be access outside of the
package. It is "exported". The name of each package level entity is visible
not only throughout the source file  that contains its declaration but
throughout all the files of the package.

Types:

Basic Types
numbers, strings booleans

Aggregate Types
arrays, structs

Reference Types
pointers, slices, maps, functions, channels

Interface Types
